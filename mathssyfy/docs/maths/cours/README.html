Inverse d'une matrice carrée

Une matrice carrée AAA d'ordre nnn est inversible s'il existe une matrice carrée BBB d'ordre nnn telle que AB=BA=IAB=BA=IAB=BA=I.
La matrice BBB, notée A−1\boldsymbol{A^{-1}}A−1, est appelée la matrice inverse de AAA.
